# Full-Adder
This repository contains the RTL design and simulation of a 1-Bit Full Adder implemented using Verilog HDL.
A Full Adder is a combinational digital circuit that adds three 1-bit binary inputs:
A – First operand
B – Second operand
Cin – Carry input
It produces:
Sum – Result bit
Cout – Carry output
# Truth Table
